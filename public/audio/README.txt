For the audio-reactive text effects to work, please add an MP3 file named "soundtrack.mp3" to this folder.

The soundtrack will be analyzed in real-time to create dynamic text animations synced to the beat. 
For best results, choose a track with a clear beat and varied frequency content.

Recommended track format:
- MP3 format
- 44.1kHz sample rate
- Stereo
- Length: 1-3 minutes
